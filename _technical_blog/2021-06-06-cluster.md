Checklist for Remote cluster working 
1. Reset password
1. Use key instead of password to save time
1. Set mobaxterm
1. Set .ssh/config file
    1. Set login alias
    1. Set to skip gateway
    1. Set port forward
    1. Set port tunnel
    1. Set x11 forwarding
